夏のLA

夏のLA開催概要

  • 日程: 2016年 7月 19日 (火) 〜 21日 (木)
  • 場所: かんぽの宿 大和平群　〒636-0905 奈良県生駒郡平群町上庄2-16-1　（詳しい交通案内は下部にございます。）

スケジュール

一般セッション 20分(発表17分＋質疑応答3分) 、学生セッション 12分(発表10分＋質疑応答2分)
（ただし、発表件数により多少変更になる場合があります。）

    • 7月19日 午後 講演
    • 7月20日 午前 講演、午後 自由討論 、夜 総会および懇親会
    • 7月21日 午前 講演

講演者の方へ

各講演の予稿を会場で配布する形にしたいと思います。 つきましては、 講演者の方は、予稿を50部印刷してご持参ください。 ご協力のほど、よろしくお願いいたします。

予稿は以下の書式でお願いいたします。

  • 用紙サイズは A4 をお使いください。
  • 1ページ目の左肩に 「2016年度夏のLAシンポジウム [発表番号]」 のように記入してください。 （発表番号が[14]の例： 2016年度夏のLAシンポジウム [14]） 発表番号はプログラムに記載されております。
  • ページ番号は「発表番号-ページ」の組でページ下中央に表記してください。 （発表番号が[14]の例： 14-1, 14-2, 14-3, …）

LaTeX 用の予稿のテンプレート （LAtemplate.tex）がございますので、 どうぞご利用ください。 5行目の「14」という数字をご自分の講演番号に書き換えていただくことで、 指定の書式の予稿を作成することができます （出力例：LAtemplate.pdf）。

かんぽの宿大和平群ホームページの交通案内も併せてご覧ください。

昼食

昼食は各自でお願いします。なお、かんぽの宿大和平群内にもレストランがあります（メニュー）。元山上口駅周辺には飲食店やコンビニエンスストアはございません。かんぽの宿のレストランや近鉄生駒駅周辺で食事をされることをお勧めします。

2日目午後

7月20日(水)午後に、ガイド付き散策（無料）を提供します。御希望の方は、参加申し込み時にコースをお選びください。

  • 山里の道コース（約2時間）
    宿→紀氏神社→蔵城薬師塚→吉備内親王墓→長屋王墓→宿
  • 石仏の里コース（約3時間）
    宿→生駒山口神社→清滝石仏群→ゆるぎ地蔵→千光寺→金勝寺→宿
  • 法隆寺コース（約3時間, 電車代が往復1,000円程度、拝観料が1,500円掛かります）

天候次第では、中止になる場合があります。予めご了承ください。

オンラインの参加登録ページよりご登録ください（申し込みは締め切りました）。ご参加の申し込みは6月15日（水）までにお願いいたします。 7月19日、20日ともに 1泊2食（夕・朝）を単位として、 以下の要領で参加登録を受け付けます。 部屋割りは事務局にお任せください。

  • 19、20日両日宿泊 24,000円 　（19日の夕食、20日の懇親会及び20日と21日の朝食を含む）

  • 20日のみ宿泊 13,000円 （20日の懇親会と21日の朝食を含む）
