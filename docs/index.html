LAシンポジウムとは

LAシンポジウムは，理論計算機科学の研究者がこの分野のテーマについて自由な議論をするための組織です． 1970年から毎年夏と冬にシンポジウムを開き，そのシンポジウムを中心に活動しています． 1985年からは会誌も年2回発行しています．
